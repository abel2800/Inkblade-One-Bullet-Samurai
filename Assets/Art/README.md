# Art Assets

This folder contains all visual assets for the game.

## Placeholder Sprites

For development and testing, you can generate simple placeholder sprites:

1. Open Unity Editor
2. Go to menu: `Inkblade > Generate Placeholder Sprites`
3. Click "Generate All Placeholders"

This will create:
- **Player** (Blue square, 64x64)
- **Enemy** (Red square, 48x48)
- **Bullet** (White square, 16x16)
- **Wall** (Gray rectangle, 32x128)
- **Ground** (Dark gray rectangle, 256x32)

## Final Art Assets

Replace placeholders with final Sumi-e ink-style assets:
- Black & white silhouettes
- Brush stroke effects
- Minimal, elegant design

## Folder Structure

- `Characters/` - Player and enemy sprites
- `Weapons/` - Bullet and weapon sprites
- `Effects/` - Particle effect sprites
- `UI/` - UI elements and icons
- `Backgrounds/` - Background textures and tiles
